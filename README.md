# books-manager
book-manager is a minimalist CRUD web app for managing books, built with React, Node, Express, and SQL

# Tools

**Front**
-React
-Sass
-Redux toolkit
-Typescript
-react-router-dom
-react bootstrap

**Backend**
-Node
-Express
-mysql
